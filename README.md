# linux-systemadmi-script
A collection of Linux administration and security scripts.

## Scripts

### audit_users.sh
Audits all system users, sudo privileges, and login history.

**Usage:**
bash audit_users.sh

**What it does:**
- Lists all human users (UID >= 1000)
- Shows sudo group members
- Displays recent login history
